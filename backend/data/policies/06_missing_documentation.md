# Policy: Missing Documentation Policy

**Policy ID:** POL-006  
**Effective Date:** January 1, 2024  
**Category:** Documentation  
**Priority:** Standard

## 1. Required Documentation by Case Type

### Refund Requests
- Proof of purchase (receipt, order ID, or transaction confirmation)
- Reason for refund request

### Damaged Goods Claims
- Photographic evidence (minimum 2 photos showing damage)
- Original packaging photo (if applicable)
- Order confirmation
- Damage description

### Fraud Disputes
- Transaction details
- Customer identification verification
- Timeline of events

## 2. NEED_MORE_INFO Protocol

When documentation is missing or incomplete:
1. Decision status is set to **NEED_MORE_INFO**
2. Specify exactly which documents are missing
3. Customer is given a **14-day grace period** to provide the missing items
4. A reminder is sent at day 7 and day 12

## 3. Grace Period Rules

- Standard customers: 14 calendar days
- VIP customers: 21 calendar days
- After the grace period expires, the case is automatically denied
- One grace period extension (7 days) may be granted upon request

## 4. Partial Documentation

If partial documentation is provided:
- The case may proceed to preliminary review
- A preliminary decision may be issued with conditions
- Final decision is contingent on complete documentation

## 5. Documentation Verification

- All submitted documents are verified for authenticity
- Altered or falsified documents trigger fraud review (POL-003)
- Documents must be legible and relevant to the claim
